ABC Retail E-Commerce Platform

Project Overview
A cloud-based solution built with ASP.NET Core MVC and Azure services, featuring user authentication, product management, shopping cart functionality, and order processing.

Technology Stack
Frontend: ASP.NET Core MVC, Bootstrap 5, HTML5

Backend: .NET 9, Entity Framework Core

Database: Azure SQL Database

Cloud Services: Azure App Service, Azure Functions, Azure Storage

Authentication: Session-based with role management (Admin/Customer)

Key Features
User registration and authentication with role-based access

Product catalog with search and categorization

Shopping cart management

Order processing with status tracking

Admin dashboard for user, product, and order management

Azure Storage integration (Tables, Blobs, Queues, Files)

Serverless order processing with Azure Functions

Azure Services Used
Azure App Service: Web application hosting

Azure SQL Database: User and order data storage

Azure Functions: Background order processing

Azure Table Storage: Product catalog management

Azure Blob Storage: Product image storage

Azure Queue Storage: Asynchronous message processing

Azure File Storage: Document and contract storage
